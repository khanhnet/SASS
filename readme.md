## Hướng dẫn cài đặt

- Chạy npm install
- Chạy npm run watch hoặc npm run prod

- Kiến thức về scss (cú pháp, biến, function, mixin, ...)
- Cấu trúc thư mục scss (7-1 pattern, SMACSS)
- Webpack và sử dụng laravel mix để compiler scss
- Đặt tên class theo BEM
